# CandyTech - Advanced Fintech & VTU Platform

CandyTech is a high-performance Laravel-based fintech application for automated VTU, Data, Bill Payments, and virtual wallet management.

## 🚀 Key Features (Phase 2)
- **Advanced Pricing Engine**: Database-driven retail pricing with universal commission rules and wholesale sync.
- **WhatsApp Notification Engine**: Automated transaction alerts, PIN delivery, and admin broadcasts via Termii/Sendchamp.
- **Auto-Subscription Logic**: "Set & Forget" recurring payments for airtime and data bundles.
- **Secure Virtual Accounts**: Instant wallet funding via Monnify dedicated virtual accounts.
- **Enhanced Security**: Transaction PINs, BVN verification, and Google 2FA support.

## 🏗 Technology Stack
- **Backend**: Laravel 11+
- **Database**: MySQL / MariaDB
- **Payments**: Monnify API
- **Service Provider**: VTPass API
- **Notifications**: Termii / Sendchamp (WhatsApp)
